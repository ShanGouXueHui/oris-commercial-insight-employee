# Insight Rebuild Module 2 - Domain Contracts

## Objective

Define the product-side domain contracts that future evidence ingestion and brief generation modules will use.

## Contracts Added

- Insight subject contract
- Evidence source contract
- Evidence item contract
- Brief section contract
- Insight domain contract
- Evidence validation against quality gates

## Required Brief Lenses

- company profile
- market structure
- competitor landscape
- financial quality
- product capability
- strategy signals
- risks
- scenarios
- limitations

## Runtime v2 Alignment

This module remains deterministic and local. It prepares product contracts that can later be executed through the Runtime v2 worker, executor, evidence publisher, and approval gate.
